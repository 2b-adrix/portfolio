# Aditya Kumar Mishra — Portfolio (Vite + React + Tailwind)

This is a ready-to-run starter for Aditya's professional portfolio.

## Quick start

1. Install dependencies:
   ```
   npm install
   ```

2. Run dev server:
   ```
   npm run dev
   ```

3. Open http://localhost:5173

## Notes
- Replace `/public/images/profile.jpg` with your actual profile photo.
- Replace `/public/resume.pdf` with your resume PDF file (the repo includes a placeholder file).
- Customize the site from `src/data/profile.js`.
