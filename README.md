A high-performance, immersive portfolio designed for a **Native Android Architect**. This project mimics the aesthetics of modern Android development environments, featuring a production-ready Gradle-themed preloader, interactive 3D models, and a highly responsive layout optimized for all devices.

---

## 🚀 Elite Features

- **Gradle-Themed Preloader**: An authentic terminal-style simulation of an Android build process (`:app:compileKotlin`, etc.) that sets the tone immediately.
- **Interactive 3D Android Mascot**: A custom-lit, hardware-accelerated 3D Android robot powered by **React Three Fiber** and **Three.js**.
- **Adaptive Ambient UI**: A dynamic theme system with layered "glass" cards and ambient background orbs that respond to light/dark modes.
- **Next.js Server Actions Contact Node**: Secure contact form that bypasses client-side ad-blockers using Next.js Server Actions and Web3Forms.
- **Performance Optimized**: Capped DPR, GPU-accelerated blurs, and adaptive event listeners to ensure a fixed 60FPS experience even on mobile devices.
- **Glassmorphic Bento Grid**: A flexible layout showcasing professional skills and project lifecycle in a modern, organized structure.

## 📥 Getting Started

### Prerequisites
- Node.js (Latest LTS)
- npm or pnpm

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/2b-adrix/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file in the root directory and add your Web3Forms Access Key:
   ```env
   NEXT_PUBLIC_WEB3FORMS_KEY=your_access_key_here
   ```
   *Get your free key at [web3forms.com](https://web3forms.com/)*

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   The site will be live at `http://localhost:3000`.
